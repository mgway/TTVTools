TTV Bonk Counter

A simple overlay label for counting the number of users banned during the current streaming session

What is this?

Ever wanted to keep track of how many chatters you ban in a session? Better yet, want to add that number to an overlay like a badge of honor?
Well, look no further! This simple OBS Browser Source compatible label adds just that functionality.

Moderation Permissions?!

This widget requires the "moderation:read" API scope which allows it to read the list of banned users and moderators. It does not have the ability to read chat messages or ban users.
Additionally, the Twitch token that is used is only ever presented to you in the browser; our servers never see it.
